# Мобильное приложение IRL Quest

## Стек
- Kotlin, Jetpack Compose
- Android

## Модули
- Квесты и задачи
- Боссы и бои
- Фокус-сессии (Pomodoro)
- Учёба и тесты
- AR/Геолокация
- Кооператив и инвентарь

## Архитектура
Модульная структура, DI через Hilt/Koin, MVVM.  
Взаимодействие с backend через защищённые API.
